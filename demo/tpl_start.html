SPARQL Micro-Services Demo

Biodiversity Data Integration

This page demonstrates the use of SPARQL micro-services [1][2] to integrate, within a single SPARQL query, information about a taxon from multiple biodiversity sources:

  • taxonomic information from the TAXREF-LD Linked Data taxonomic register [3],
  • occurrences from the Global Biodiversity Information Framework (GBIF),
  • photos from Flickr,
  • articles from the Biodiversity Heritage Library,
  • traits from the Encyclopedia of Life trait bank, and
  • audio recordings from the Macaulay Library.

To test with another taxon, simply enter the taxon name (without authority) as parameter param in the page URL. The taxon must be present in TAXREF, hence it must have been inventoried in French territories. For instance enter Delphinus delphis or Aegolius acadicus (loading the page may take a few seconds).

The SPARQL results are transformed into HTML using the SPARQL Template Transformation Language [4]. The code of this demo is available at https://github.com/frmichel/sparql-micro-service/tree/master/demo.
